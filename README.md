# RU:

Расширение для твиттера, позволяющее просматривать и скачивать любые изображения (в будущем будет добавлена поддержка видео).

_Установка_:
1. Открыть настройки расширений вашего браузера
2. Через пункт "Добавить распакованное расширение" выбрать папку с расширением

_Использование_: 
1. Перейти на страницу твита
2. Открыть меню расширения (нажатием на его иконку)
3. В меню будет список изображений. При наведении на изображение можно посмотреть уменьшенную версию

P.S. расширение делалось под мои требования, но в будущем я планирую сделать интерфейс более дружелюбным к пользователю


# EN:

A Twitter extension that allows you to view and download any images (video support will be added in the future).

_Installation_:
1. Open your browser's extension settings.
2. Select the folder with the extension via “Add unpacked extension.”

_Usage_:
1. Go to the tweet page.
2. Open the extension menu (by clicking on its icon).
3. The menu will display a list of images. Hovering over an image will show a thumbnail version.

P.S. The extension was made to my specifications, but I plan to make the interface more user-friendly in the future.
